## Description

```markdown
A movable panel showing your secondary stats in your specialization's priority
order, alongside item level, movement speed and framerate.

Pick one of 21 presets and you're done — or open the options and change
essentially every pixel: colors, textures, borders, fonts, sizes, layout, which
stats appear, and how their numbers are written.

## Built for patch 12.0

Midnight's "secret values" hand most combat stats to addons in a form that can
be displayed but not read. StatPanel is built for this and shows your stats
normally. The few features that can't work on a protected value degrade quietly
instead of erroring.

## What's in it

**Stats** — Crit, Haste, Mastery and Versatility in spec priority order; armor
damage reduction, dodge, parry, block, leech and avoidance; movement speed with
a session peak that tracks ground, flight, swimming and skyriding; primary
attributes. Rename sections, reorder them, move any stat between them.

**Looks** — Status bars or a text-only readout. Per-stat colors, class color, or
a value gradient. Auto-sizing or fixed width. Two presets read your live ElvUI
theme and match it.

**Profiles** — Per character, with import/export strings for sharing. Switch
automatically by specialization or by content type (raid, dungeon, arena,
battleground, open world).

**Extras** — Right-click menu for quick toggles, minimap button, LibDataBroker
feed, and a gear audit (`/sp gear`) that flags missing enchants, empty sockets
and per-slot item level.

## Getting started

Type `/sp` to open the options. Drag the panel to move it.

No dependencies. If any addon you run provides LibSharedMedia-3.0 — ElvUI,
WeakAuras, Details!, Plater — its textures and fonts appear in StatPanel's
dropdowns automatically.
```
